# LeetCode Review System

A Java application that helps users track solved LeetCode problems, manage review schedules, and build consistent coding habits.

## Features (MVP)

- Add solved LeetCode problems
- Store problem information
- Schedule review dates
- Track progress
- View statistics

## Tech Stack

- Java
- Object-Oriented Programming (OOP)
- File Storage (CSV)

## Future Features

- Spaced Repetition
- GUI
- User Login
- Mobile App
- Daily Streak
- Achievement System

## Why I Built This

I wanted a lightweight tool to help software engineering interview preparation by tracking solved problems and review schedules.
